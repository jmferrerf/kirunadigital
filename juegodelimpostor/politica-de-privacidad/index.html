Política de Privacidad de El Juego del Impostor

Última actualización: 30 de enero de 2026

Esta Política de Privacidad describe cómo Kiruna Digital ("nosotros", "nuestro") gestiona la información en la aplicación móvil El Juego del Impostor.

1. Recopilación de Información

Nuestra aplicación está diseñada para ofrecer una experiencia de juego sencilla y segura.

  • Información Personal: No recopilamos, almacenamos ni solicitamos ninguna información de identificación personal (como nombre, dirección, correo electrónico o número de teléfono) para utilizar la aplicación.
  • Información No Personal: La aplicación no utiliza herramientas de análisis de terceros que rastreen tu actividad de forma individual.

2. Publicidad

La aplicación El Juego del Impostor no incluye publicidad de ningún tipo. No compartimos identificadores de publicidad de tu dispositivo con anunciantes ni con redes de terceros.

3. Permisos del Dispositivo

Para el correcto funcionamiento del juego, la aplicación podría solicitar permisos básicos (como acceso a internet para cargar el contenido de la web o vibración para notificaciones del juego). Estos permisos se utilizan exclusivamente para la funcionalidad técnica de la aplicación y nunca para extraer datos privados.

4. Privacidad de los Niños

Dado que El Juego del Impostor es un juego apto para todas las edades, cumplimos con las normativas de protección de menores (como COPPA y GDPR). Al no recopilar ningún tipo de dato personal, la aplicación es segura para que la utilicen menores de edad.

5. Proveedores de Servicios

Dado que la aplicación es una versión de nuestra plataforma web, es posible que se recojan datos técnicos estándar (como dirección IP o tipo de dispositivo) de forma automática por los servidores de alojamiento para garantizar la estabilidad del servicio, pero estos datos no son accesibles para nosotros con fines comerciales ni de identificación.

6. Cambios en esta Política

Podemos actualizar nuestra Política de Privacidad de vez en cuando. Se recomienda revisar esta página periódicamente para ver si hay cambios. Cualquier modificación entrará en vigor inmediatamente después de su publicación en este enlace.

7. Contacto
